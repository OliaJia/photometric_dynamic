This folder contains the frog and scholar datasets used in the paper 

Visibility Subspaces: Uncalibrated Photometric Stereo with Shadows
Kalyan Sunkavalli, Todd Zickler, Hanspeter Pfister
European Conference on Computer Vision, 2010

Each dataset includes the images used to recover the surface normals using our method, as well as the calibrated light positions and "true" normals recovered by robustly detecting and discarding shadowed pixels from a dense set of images. For more details, please refer to the paper.
